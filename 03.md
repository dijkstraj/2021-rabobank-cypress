
# Introduction: what is Cypress?

* Web testing (UI _and_ API)
* Synchronous code
* No Selenium/WebDriver
* Supported browsers:
    * Chrome
    * Firefox
    * Electron
    * Edge
    * Brave
* TL;DR: `npm install --save-dev cypress && npx cypress open`
